# WikiViewer - Εφαρμογή Προβολής Πληροφοριών Wikipedia 📚

Εφαρμογή Java για την ανάκτηση, προβολή και διαχείριση άρθρων από την ελληνική Wikipedia.  
Το έργο υλοποιείται στο πλαίσιο της Θεματικής Ενότητας **ΠΛΗ24 «Σχεδιασμός Λογισμικού»** του ΕΑΠ (2025-2026).

## 🛠 Τεχνολογίες & Εργαλεία
**IDE:** NetBeans  
**Γλώσσα:** Java  
**Βάση Δεδομένων:** Apache Derby  
**Σύνδεση με ΒΔ:** Java Persistence API (JPA)  
**API:** MediaWiki API (REST/JSON)  
**Σχεδιασμός:** Visual Paradigm (UML/ER)  
**Διαχείριση Έργου:** Scrum μέσω Trello  

## 📂 Δομή του Repository
*`/WikiViewerApp`: Ο πηγαίος κώδικας του project στο NetBeans.  
*`/documents`: Έγγραφα διαχείρισης έργου (Backlog, Χρονοδιάγραμμα) και διαγράμματα (ER, Class Diagrams).  
*`/database`: Το script δημιουργίας και τα αρχεία της βάσης Derby.  
*`/javadoc`: Αυτόματα παραγόμενη τεκμηρίωση.
