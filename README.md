# WikiViewer - Εφαρμογή Προβολής Πληροφοριών Wikipedia 📚

Μια desktop εφαρμογή Java για την ανάκτηση, προβολή και διαχείριση άρθρων από την ελληνική Wikipedia.  
Το έργο υλοποιήθηκε στο πλαίσιο της Θεματικής Ενότητας **ΠΛΗ24 «Σχεδιασμός Λογισμικού»** του ΕΑΠ (2025-2026).  

> 💡 **Σημείωση:** Η εφαρμογή αναπτύχθηκε με χρήση Agile/Scrum μεθοδολογίας και πλήρη διαχωρισμό επιπέδων (UI, Λογική, Βάση Δεδομένων).

---

## ✨ Βασικές Λειτουργίες
* Ζωντανή αναζήτηση και ανάκτηση άρθρων μέσω του **MediaWiki REST API**.
* Τοπική αποθήκευση και διαχείριση αγαπημένων άρθρων.
* Προβολή ιστορικού και στατιστικών.
* Εξαγωγή δεδομένων και αναφορών σε μορφή PDF.

## 🛠 Τεχνολογίες & Εργαλεία
* **IDE:** NetBeans
* **Γλώσσα:** Java (Swing για το GUI)
* **Βάση Δεδομένων:** Apache Derby (Embedded)
* **ORM:** Java Persistence API (JPA / EclipseLink)
* **Διαδίκτυο:** MediaWiki API (REST / JSON)
* **Σχεδιασμός:** Visual Paradigm (UML / ER)
* **Διαχείριση Έργου:** Scrum μέσω Trello

---

## 🚀 Οδηγίες Εγκατάστασης & Εκτέλεσης
Για να τρέξετε την εφαρμογή τοπικά στον υπολογιστή σας:
1. Κάντε clone το repository: `git clone https://github.com/SakisAA/WikiViewer.git`
2. Ανοίξτε τον φάκελο του project (`/WikiViewerApp`) στο **NetBeans**.
3. Βεβαιωθείτε ότι έχετε κάνει **Resolve** τις βιβλιοθήκες στον φάκελο `lib`.
4. Εκτελέστε **Clean and Build**.
5. Πατήστε **Run** για να ξεκινήσει το γραφικό περιβάλλον.
